# VMSDEP Templates Extension

A browser extension for building standardized VMSDEP email responses using selectable templates, dynamic greetings, and reusable template sections.

## Purpose

This project is designed to help create consistent VMSDEP email responses more quickly while keeping the wording organized and easy to update.

## Main Features

- Select one or more email templates
- Combine selected templates into one response
- Add a dynamic greeting
- Add a standard closing
- Copy the finished response for use in email
- Store template wording in a separate `templates.json` file

## Project Files

- `manifest.json` — extension settings and permissions
- `popup.html` — extension popup layout
- `popup.js` — extension behavior and template logic
- `styles.css` — popup styling
- `templates.json` — stored email templates
- `icons/` — extension icons and logo files
